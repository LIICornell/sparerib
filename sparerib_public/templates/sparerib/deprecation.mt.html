Deprecation

In response both to improvements in Regulations.gov, Docket Wrench's primary data source, and to Sunlight Labs staff transition, the Labs team has decided to retire the Docket Wrench project and focus its efforts on new initiatives.

The deprecation timeline for this project is as follows:

  • Beginning August 28th, 2015, updates to the underlying data may become intermittent and unpredictable.
  • As of September 30th, 2015, data updates will likely cease altogether, and the site and its API will continue to be available, but serving only the static data as of their last update.
  • Sunlight will make a best effort to keep the site and its API available in that frozen state until December 1st, 2015, after which both will be turned off and will no longer be available.

Unforeseen circumstances (e.g., server hardware failures) may result in an acceleration of this timeline.

We want to thank you for your interest in and support of this project. Its source code will continue to be available on Github in the regulations-scraper and sparerib repositories.
